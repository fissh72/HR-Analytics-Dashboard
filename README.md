# HR Analytics Dashboard: Employee Attrition & Workforce Overview

## 📊 Project Overview
This Power BI project provides a comprehensive analysis of a corporate workforce, focusing on **employee demographics, salary distribution, and attrition trends**. The goal is to help HR departments identify high-risk departments and understand the factors driving employee turnover.

## 🛠️ Tech Stack & Tools
* **Power BI Desktop:** Dashboard creation and data visualization.
* **Power Query (M):** Data cleaning, transformation, and date table generation.
* **DAX (Data Analysis Expressions):** Calculated measures for Attrition Rate, Active Employees, and Average Salary.
* **Figma/Power BI Shapes:** For the custom UI/UX sidebar navigation.

## 📈 Key Insights & Features
* **Executive Overview:** At-a-glance KPIs for Total Headcount (311), Average Salary ($69K), and Attrition Rate (50.24%).
* **Demographic Breakdown:** Interactive analysis of the workforce by Gender, Citizenship, and Marital Status.
* **Attrition Deep-Dive:** Identification of the top reasons for leaving, with "Another Position" and "Unhappiness" as primary drivers.
* **Departmental Analysis:** Visualizing that the **Production Department** holds the majority of the workforce (209 employees) but also faces significant turnover challenges.
* **Dynamic Navigation:** A custom sidebar allows users to toggle seamlessly between the Overview, Attrition, and Employee Detail views.

## 📂 Data Structure
The dataset includes:
* **Employee Demographics:** Age, Sex, Race, Marital Status.
* **Employment Details:** Department, Position, Date of Hire, Manager.
* **Attrition Data:** Termination Date and Termination Reason.
